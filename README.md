# Shital Bhelave — Portfolio

## Folder Structure
```
portfolio/
├── index.html          ← Main file
├── assets/
│   ├── css/
│   │   └── main.css    ← All styles
│   ├── js/
│   │   └── main.js     ← Interactions, typed effect, nav
│   └── img/
│       └── my-profile-img.jpg  ← ADD YOUR PHOTO HERE
```

## Setup

1. **Add your photo:**
   - Copy your profile photo to `assets/img/my-profile-img.jpg`

2. **Run locally:**
   - Simply open `index.html` in a browser
   - OR use VS Code Live Server extension

3. **Deploy to Netlify:**
   - Go to https://app.netlify.com
   - Drag & drop the entire `portfolio/` folder onto Netlify Drop
   - Done! 🚀

## Features
- Dark sidebar with your photo + name
- Typing animation (hero section)
- Skills organized by category (Frontend / Backend / DB / DevOps)
- 5 projects from your resume (EWS, WMS, TBS, PMS, Low-Code)
- Timeline resume (experience + education)
- Contact form (opens email client)
- Scroll animations (AOS)
- Fully responsive (mobile friendly)
- Active nav highlight on scroll
